## 创建一个新文件 ## 
> hexo new "新博客文件名"

## 清空之前主题 ## 
> hexo clean 

## 在本地运行博客 ## 
> hexo s / hexo server

## 生成以及部署文章之前，需要安装一个扩展 ## 
> npm install hexo-deployer-git --save

## 生成静态文件 ## 
> hexo g  / hexo generate

## 发布到github的page上 ## 
> hexo d  / hexo deploy

## 生成静态文件并发布 ## 
> hexo d -g
